Detailed Description

Single animation clip on a spritesheet.

Defines the source rectangle, frame count, timing and texture for one animation clip. Multiple clips are stored in Animator and switched via Animator::play.

Note
startX and startY are in frames, so startX = 0 and startY = 1 means first frame from second row, multiply by frameWidth and frameHeight to get pixel offsets
