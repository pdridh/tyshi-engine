Detailed Description

Manages animation clips and drives animation frame advancement for an entity.

Stores a named map of Animation clips and tracks which clip is currently playing. Call update every frame to advance animation frames and play to switch clips.

Each clip can use its own spritesheet via Animation::textureSheetID or setTextureSheetForAllAnims can be used when all clips share one texture as a sheet.

◆ play()

void Animator::play ( const std::string &  id)

Switches to the animation clip with the given ID.

In debug, asserts if the given ID is not valid, in release, just logs it to stdout without changing the current animation.

Resets the frame time and current frame to 0 when the current animation clip is changed.

No-op if the clip is already playing.

Parameters
idName of the clip to play, must be registered in animations before calling
