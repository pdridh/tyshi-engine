Detailed Description

Represents a dynamic debug entry that can get removed if not updated.

If DebugOverlay::setDynamicEntry is called for this entry that frame then updated will be set to true and it won't be destroyed next frame otherwise gets destroyed by DebugOverlay::beginFrame next frame.
