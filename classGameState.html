Detailed Description

Abstract base class for all game states.

Defines the interface for the engine's state stack. States are pushed and popped by the engine — only the top state receives update and draw calls each frame.

Subclasses must implement onEnter, onExit, update and draw. pause, resume and onScreenResize are optional and default to no-ops.

Note
Constructors of subclasses should be private: use a static instance() method to enforce a single instance per state
